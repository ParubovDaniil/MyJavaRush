Под дженериками в Java подразумевают возможность добавлять к типам типы-параметры.

Компилятор просто заменяет тип с параметром на него же, только без параметра.
А при взаимодействии с его методами добавляет операцию приведения типа к типу-параметру

Т.е. по сути дженерики — это такая разновидность синтаксического сахара, как и autoboxing, только побольше.
При autoboxing компилятор за нас добавляет методы для преобразования типа int к Integer и обратно, а для generics добавляет операторы приведения типа.

После того, как компилятор скомпилировал ваш код с дженериками, в нем все классы с параметрами были преобразованы просто в классы и операторы приведения типа.
Информация о том, какие изначально были типы-параметры у переменных сложных типов, потерялась. Этот эффект еще называют стиранием типов.
